1. هيكل المشروع العام

PMS-Project/ ├── backend last/ ← Laravel 10 API │ ├── app/ ← Controllers, Services, Models, Middleware │ ├── config/ ← إعدادات (ai, sanctum, services, …) │ ├── database/migrations/ │ ├── routes/api.php ← كل REST endpoints │ └── tests/ ├── frontend/ ← React 19 SPA │ ├── src/ ← pages, components, context, utils │ ├── vite.config.js │ └── package.json └── docs/ ← ERD, architecture, guides, PDFs
النمط: Controller (HTTP) → Service (منطق) → Model/DB · Frontend: Page → Component → apiFetch → API
