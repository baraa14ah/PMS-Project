5. Authentication Flow — Sanctum

sequenceDiagram
    participant U as User Browser
    participant F as React AuthContext
    participant A as Laravel API
    participant T as personal_access_tokens
    participant D as MySQL

    U->>F: Login email/password
    F->>A: POST /api/login
    A->>D: verify user + role + status
    A->>T: createToken auth_token
    A-->>F: token + user
    F->>F: localStorage token
    F->>A: GET /api/profile Bearer token
    A-->>F: user profile
    Note over F,A: Every request uses Authorization Bearer
